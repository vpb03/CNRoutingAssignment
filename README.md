# CNRoutingAssignment

## 📘 Overview
This repository contains a Cisco Packet Tracer networking assignment covering router configuration and routing protocols. The tasks include basic router setup, static routing, and dynamic routing using RIP and EIGRP.

---

## 📁 Project Files

### Task 1 – Basic Router Configuration
- File: `Task1_BasicRouterConfig.pkt`
- Description: Basic router setup including interface IP configuration and initial connectivity.

---

### Task 2 – Static Routing Configuration
- File: `Task2_StaticRouting.pkt`
- Description: Configuration of static routes between routers to enable communication between different networks.

---

### Task 3 – Dynamic Routing Configuration (RIP & EIGRP)
- File: `Task3_DynamicRouting_RIP_EIGRP.pkt`
- Description: Implementation of dynamic routing protocols:
  - RIP (Routing Information Protocol)
  - EIGRP (Enhanced Interior Gateway Routing Protocol)

---

## 🔁 Routing Protocols Used

### RIP
Routing Information Protocol  
- Simple dynamic routing protocol  
- Uses hop count as metric  
- Suitable for small networks  

### EIGRP
Enhanced Interior Gateway Routing Protocol  
- Faster convergence than RIP  
- Uses bandwidth and delay as metrics  
- Efficient for medium and large networks
